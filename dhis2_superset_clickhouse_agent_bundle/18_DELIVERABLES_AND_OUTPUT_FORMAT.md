# Deliverables and Output Format

## Required deliverables
- Refactored code
- DDL and schema definitions
- Refresh orchestration updates
- Test suite updates
- Configuration documentation
- Migration documentation
- Rollback documentation
- Architecture summary

## Required final implementation report format
The agent should provide:

### Summary
A concise explanation of what changed.

### Files changed
A list of changed and added files.

### Architecture changes
A short section describing the new raw, staging, and serving flow.

### Migration notes
How to move from old implementation to new.

### Validation results
What was tested and what passed.

### Remaining risks
Any residual items that need future work.
