# PUBLIC-DASHBOARD-LAYOUTS.md
## Public Dashboard Layout Requirements

## 1. Goal
Public dashboards must have a professional presentation shell distinct from internal editing views.

## 2. Required Settings
Provide support for:
- full width vs contained width
- header visibility
- title/subtitle visibility
- logo/branding
- filter placement
- spacing density
- card/container style
- mobile stacking behavior
- embed / public presentation mode
- footer visibility where appropriate

## 3. Theme Interaction
Public dashboard layout settings must work coherently with the public theme layer.

## 4. Quality Standard
Public dashboards must look polished, branded, readable, and presentation-ready.
